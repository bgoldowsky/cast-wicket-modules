# Introduction #

This module contains an HTML user interface and Wicket components to allow web site users to record an audio clip, and play it back.

# Dependencies #
  * [cwm-components](CwmComponents.md)

# Details #

Audio files in the .au file format can be recorded and played back.  This module contains the HTML with associated javascript and formatting to record and playback audio files.  The actual storage and retrieval of the audio file is performed in the cwm-data module.
